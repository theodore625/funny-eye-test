# funny-eye-test
## 1. This is ***NOT*** a real eye test
I made this completely for fun
## 2. What this is
So its a eye test... but impossible. so you need to use *3d* Anaglyph glasses (**RED LENSES**) and easy mode will make it easier to see with anaglyph
Press A to lock-in your answer when you think you see the amount of rectangles
## 3. When it changes
### When you lock-it-in $${\color{red}Wrong}$$
The amount will not change.
### When you lock-it-in $${\color{green}Right}$$
The amount will change.
